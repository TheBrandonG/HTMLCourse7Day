Interesting Facts About HTML

History of HTML

HTML, which stands for HyperText Markup Language, was created by Tim Berners-Lee in late 1991. It was the first language that allowed the creation of structured documents by defining structural semantics for text such as headings, paragraphs, lists, links, quotes, and other items.

Purpose of HTML

HTML is used to create web pages and web applications. It describes the structure of a webpage and, along with CSS and JavaScript, forms a triad of cornerstone technologies for the World Wide Web. HTML elements are the building blocks of HTML pages.

Evolution of HTML

Since its inception, HTML has undergone several updates. The current version, HTML5, was published in October 2014 and includes new elements, attributes, and behaviors, as well as a larger set of technologies that allow for more diverse and powerful websites and applications.

How To

Installing Visual Studio Code

  1. Go to the Visual Studio Code website: https://code.visualstudio.com/
  2. Click on the download button for your operating system (Windows, macOS, or Linux).
  3. Once the download is complete, open the installer and follow the installation instructions.

Creating Your First HTML Document

  1. Open Visual Studio Code.
  2. Create a new folder for your project.
  3. Inside the project folder, create a new file and name it index.html.

Setting Up GitHub

Creating a GitHub Account

  1. Go to the GitHub website: https://github.com/
  2. Click on the "Sign up" button and fill in the required information to create your account.

Forking the Repository

  1. Log in to your GitHub account.
  2. Go to the HTML Course repository: https://github.com/UFLIPTraining/HTMLCourse7Day
  3. Click on the "Fork" button at the top-right corner of the page to create a copy of the repository in your own GitHub account.

Disabling the Template Repository

  1. Go to the settings of your forked repository on GitHub.
  2. Scroll down to the "Template repository" section.
  3. Uncheck the "Template repository" checkbox.
  4. Click "Save changes".

Collaborating on the Repository

  1. Go to the settings of your forked repository on GitHub.
  2. Click on the "Manage access" tab in the left sidebar.
  3. Click on the "Invite a collaborator" button.
  4. Enter the username JMartinUF and click "Add collaborator".
  5. Click "Send invitation" to invite JMartinUF to collaborate on your repository.
